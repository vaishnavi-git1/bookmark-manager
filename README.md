# bookmark-manager
Smart bookmark manager with feedback system
Bookmark Manager with Feedback System

Project Overview
This project is a simple Bookmark Manager web application. It allows users to save, search, edit, and delete bookmarks easily. It also collects user feedback when the user exits the application.

Features
- Add bookmarks with title and URL
- Search bookmarks
- Mark bookmarks as favorite
- Edit bookmarks
- Delete bookmarks
- View total bookmarks, favorites, and added today
- Data stored using local storage
- Colorful and simple user interface
- Exit button with feedback popup
- Multiple feedback options
- Custom feedback input
- Notification messages instead of alerts

Technologies Used
- HTML
- CSS
- JavaScript

Project Structure

index.html
style.css
script.js
README.md

Feedback System
When the user clicks the exit button, a popup appears asking for feedback. The user can choose from given options or write their own reason. This helps in improving the application and understanding user behavior.

Project Objective
The goal of this project is to create a simple and useful web application that manages bookmarks and collects user feedback to improve user experience.
